Подготовь датасет для дообучения модели (LoRA).

Входные данные (логи RAG: вопрос, контекст, ответ):
{{context}}

Для каждой записи:
1. Оцени качество ответа (1–5).
2. Если ответ неполный — предложи правильный ответ на основе контекста.
3. Если ответ содержит ошибку — исправь.
4. Отметь, какие уточняющие вопросы модель должна была задать, но не задала.

Выдай результат в формате для LoRA-обучения (только JSON):
[
  {"instruction": "...", "context": "...", "response": "...", "quality": 1}
]

Язык: русский.

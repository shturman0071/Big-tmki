Создай тестовый набор вопросов для оценки RAG-системы.

Документы:
{{context}}

Для каждого документа сгенерируй:
- 3 простых вопроса (факты из документа)
- 2 вопроса на понимание (связи между разделами)
- 1 сложный вопрос (требует анализа нескольких разделов)

Для каждого вопроса укажи:
- ожидаемый ответ
- источник (раздел/пункт документа)
- сложность (easy|medium|hard)

        "type": "simple|understanding|complex",
        "question": "...",
        "expected_answer": "...",
        "source": "...",

      }
    ]
  }
]
